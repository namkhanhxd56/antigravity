# Budget Friendly — Amazon Listing Skill

## Voice & Tone
Write for the practical, value-conscious buyer. Tone is direct, honest, and reassuring. Prove value through specifics — dimensions, what's included, what problem it solves — not through superlatives. The buyer is comparing options; make the decision obvious.

## Title Rules
- Lead with the primary use case or problem solved
- Include key spec that signals value (quantity, size, compatibility)
- Add a differentiator that justifies the purchase over cheaper alternatives
- Avoid luxury language. Keep it plain and searchable.
- 130–170 characters

## Bullet Rules
- Open with the practical outcome the buyer cares about
- Include concrete specs: dimensions, weight, compatibility list, quantity in pack
- Address the #1 objection for budget products (durability, fit, reliability)
- Mention what's included in the box if relevant
- 160–220 characters per bullet

## Description Rules
- Open by stating the problem this product solves
- List what you get and why it works
- Address common concerns: returns policy, compatibility, sizing
- Use plain language — no jargon, no flowery prose
- 600–900 characters

## Search Terms Rules
- Include budget/value signal terms (affordable, pack of, set of, replacement for)
- Include compatibility terms (fits, compatible with, works with)
- Include use-case terms specific to the product category
- Lowercase, space-separated, max 250 characters
